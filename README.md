# Eudaimonia Benchmark Page

Static GitHub Pages site for the Eudaimonia Benchmark public release.

Authors: Jun Rui Huang, Wang Bill Zhu, Ziyi Liu, Nathanael Fast, Ravi Iyer,
Robin Jia.

Paper: https://arxiv.org/abs/2605.30654

Dataset: https://huggingface.co/datasets/eudaimonia-bench/eudaimonia-bench

Dataset schema: `user_input`, `measure`, `synthetic`, `language`,
`rewriter_model`, `responser_model`.
